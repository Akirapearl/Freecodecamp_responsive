Introduction

Computers are probably the most important part of our daily life now a days, they provide communication with people all over the world, as well as any type of information we could need.
But, even if we had the same number of computers, those would be mostly useless without a way to communicate with each other, that channel of communication is what we know as "The internet", or just "Internet".

About the internet

The internet is not a new thing, in fact it all started back in the 1960s when the United States's defense department
started to build the ARPANET, a network that was intended to be a point of communication between universities all over
that country.
The time went on and now there are billions of computers connected to each other via huge networks, connected between each other as well.

First data transmision

When we talk about internet, most of the people think about themselves sitting in front of their computer or smart phone,
but that was not there since the beginings, in fact, it was not until the early 1990s that Tim Berners Lee
created the first client-server connection when it all started. That moment was when the actual www protocol and the internet started as well.

ping

Now you can see two type of console commands, the first one is intended to check if two diferent nodes "see" each other, so they can send messages.
The other two are used on Windows and Gnu/Linux systems respectively to see the IP of the current pc/device the user is at.
The IP is just a identification number of the device, intended to be used through different types of communication through networks and the internet itself.

The browser wars

But...how do actually two nodes (one server and one client on each point) communicate with each other?.
We use a specific type of program called Browser, a browser is intended to be an interpreter of different
languages, all of them written with a different purpose, HTML being the basic and probably the most important one among
all of the current existent ones.

  • Html
  • Css
  • Javascript
  • Php
  • Learn more about web developing...
html_example css_sample

HTML stands for hypertext markup language, its the most basic language that a browser can read
A web developer who uses html needs to create a generic structure inside what they will use differnt tags
inside "<>" characters, each of those with a different function.

CSS stands for cascade style sheet, it is intended to be a language that gives more functionality to a plain
and basic html file, it can be used to set colors, sizes, even animation or responsive behavior for different screen sizes!

The above mentioned languages are part of what we call frontend, it can be defined as the part of the web that the user see
at their screen when they look for any particular website.

Present and future

We have covered a little about the internet at the basic languages that are used to display a website but, how about the actual
workarounds, what do developers use to create their sites? is anything new and revolutionary coming to scene?

Well, in fact, despite html and css are enough to build a "basic" site, now a days most of the developers use different frameworks that help them
to create more complex and advanced sites within less time writing code, that has evolved into new trends called "nocode" that consist as they say, writing
websites with no code, just graphical edition with drag and drop elements, with all the lines of code being interpreted on the background.
